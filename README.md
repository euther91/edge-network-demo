## Usage

```sh
make install
```

## Run Cloudflare Worker Function (wrangler)

```sh
make cloudflare-dev
```

Then open at http://localhost:8787 on your browser.

## Run Netlify Edge Functions

```sh
make netlify-dev
```

Then open http://localhost:8888/test on your browser.
